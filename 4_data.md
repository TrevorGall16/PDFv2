## 4. Dynamic Shopping List Engine
* **Logic**: Only ingredients from active (non-deleted) rows are extracted.
* **Interactivity**: Fully editable, includes `[+ Add Item]` and `[x]` delete buttons.
* **Output**: Page 2, clean checkbox list, zero-ink design.
* **Ingredient Management**: 
    - Ingredients are extracted from active swaps.
    - Users can delete specific ingredients manually via an [x] icon next to each checkbox.
    - New items added via [+ Add Item] are fully integrated into the list.

    * **Tiering**: Items 3, 4, and 10 remain the "Tier 1" defaults but are fully swappable via the "Pin" feature.
* **Notes**: Every item has an editable "Strategy/Notes" field instead of a score.
